Bootbox - extend, adapt and customize Bootstrap

Bootbox is a collection of leightweight extensions and additions for the base Bootstrap styles, intended to make them more appealing for use in styled projects. This Page is intended to showcase the different additional styles you can include.

To include these Styles in your own project, follow the documentation in the github-repository.
